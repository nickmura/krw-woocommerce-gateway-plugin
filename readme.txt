Accept KRW stablecoin payments in WooCommerce.

== Description ==

A payment gateway for WooCommerce that enables KRW stablecoin payments with wallet redirect functionality.

== Installation ==

1. Upload plugin files to `/wp-content/plugins/woocommerce-krw-gateway/`
2. Activate the plugin
3. Go to WooCommerce -> Settings -> Payments
4. Enable "KRW Stablecoin"
